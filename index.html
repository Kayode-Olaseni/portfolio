Fuzzy C-Means News Articles Clustering
with python

This research project developed a clustering technique using the fuzzy c-means algorithm with N-grams for similarity measurement. Unlike K-means, which assigns documents to a single cluster, fuzzy c-means allows overlap, addressing cases where documents belong to multiple clusters. The project tested different N-gram window sizes, with the proposed method achieving 48.51% accuracy and 55.09% purity, showing a 13.93% improvement in accuracy over the baseline.
